# Week 01 — Introduction to Algorithms

## Overview
- **Learning Objectives**:
  - Understand the course structure and grading policy
  - Grasp what an algorithm is and how it differs from a data structure
  - Experience classic introductory algorithm problems (search, greedy, divide-and-conquer, binary encoding)
