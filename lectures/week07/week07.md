# Week 07 — Review and Problem Solving

## Overview
- **Learning Objectives**:
  - Consolidate all topics from Weeks 01–06 before the midterm exam
  - Compare algorithm design paradigms (Divide & Conquer, Greedy, DP) and know when to apply each
  - Practice representative exam-style problems with full solutions
  - Identify personal weak areas and form a study plan for the midterm

## Class Schedule

  - Algorithm definition, properties, representation (pseudocode)
  - Classic introductory problems (binary search, coin change, Euler path)
- [00:25–00:35] Review: Week 02 — Algorithm Design and Complexity Analysis

- [00:35–00:50] Review: Week 03 — Data Structures and Sorting

  - Selection sort, Bubble sort, Insertion sort — O(n^2)
  - Merge sort, Quick sort, Heap sort — O(n log n)
  - Radix sort, Counting sort — O(n)
