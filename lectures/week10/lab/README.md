## 프로젝트 작업 (25분)

### Proj 1: 해시 기반 기능 추가 (15분)
프로젝트에 해시 테이블을 활용한 기능을 추가합니다:
- 쇼핑몰: 상품 ID → 상품 정보 빠른 조회, 장바구니 캐싱
- 소셜 네트워크: 유저 ID 매핑, 세션 관리
- 캠퍼스 지도: 건물 이름 → 좌표 매핑
