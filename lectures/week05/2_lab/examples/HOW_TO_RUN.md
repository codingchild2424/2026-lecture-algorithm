# Week 05 Lab Examples -- How to Run

## Skeleton Files (Student Versions)

Each `.py` file in this directory contains skeleton code with `TODO` comments.
Implement the missing function bodies, then run to verify your solution.

```bash
# A-1: Coin Change (greedy + DP)
python a1_coin_change.py

# A-1: Coin Change -- Greedy vs DP detailed comparison
python a1_coin_change_greedy.py

# A-2: Fractional Knapsack
python a2_fractional_knapsack.py

# A-3: Huffman Coding
python a3_huffman.py
```

## Reference Solutions

Complete solutions are available in the `solutions/` subdirectory:

```bash
python solutions/a1_coin_change.py
python solutions/a1_coin_change_greedy.py
python solutions/a2_fractional_knapsack.py
python solutions/a3_huffman.py
```
