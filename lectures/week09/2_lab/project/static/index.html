Algorithm Review Web App

Weeks 01-07 in one interactive demo: Sorting, Binary Search, Greedy, and DP

Sorting Algorithms

Enter a list of numbers (comma or space separated) and choose an algorithm. View the step-by-step process and compare all algorithms at once.

Performance Comparison

Click Compare All above to run every sorting algorithm on the same input and see a side-by-side breakdown.

Binary Search

Enter a list of numbers and a target value. The array will be sorted automatically and binary search will show each step.

Greedy Algorithm -- Coin Change

Given an amount and a set of coin denominations, the greedy approach repeatedly picks the largest coin that fits. This is optimal for standard denominations but can fail for arbitrary coin sets.

Fibonacci -- Naive Recursion vs DP

Compare how many function calls naive recursion needs versus bottom-up dynamic programming for the n-th Fibonacci number.

0/1 Knapsack Problem

Enter items (one per line: name,weight,value) and a knapsack capacity. The DP algorithm finds the maximum value subset that fits.
