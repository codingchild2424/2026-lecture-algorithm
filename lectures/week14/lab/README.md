# Week 14 — 프로젝트 발표

## 발표 진행

### 형식
- 팀당 10분: 발표 7분 + Q&A 3분
- 모든 팀원이 발표에 참여

### 발표 내용
1. 프로젝트 소개 (웹앱 주제, 기능 설명)
2. 적용한 알고리즘 설명 (최소 4개)
3. 성능 비교 (before/after 측정 결과)
4. 라이브 데모
5. 어려웠던 점 & 배운 점

### 제출물
- GitHub 저장소 링크 (코드 + README)
- 발표자료 (PDF 또는 슬라이드)
