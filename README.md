# 2026 Algorithms

Korea University Sejong Campus, Department of Computer Science

## Course Structure

- **Weeks 01-07**: Foundations (Introduction, Complexity, Sorting, D&C, Greedy, DP, Review)
- **Week 08**: Midterm Exam
- **Weeks 09-14**: Advanced Topics (Search Trees, Hashing, Graphs, NP-Completeness, Approximation)
- **Week 15**: Final Exam

## Materials

- Textbook: Introduction to Algorithms, 3rd Edition (CLRS)
- Slides: Slidev (Markdown-based presentations)

## Grading

| Component | Weight |
|-----------|--------|
| Assignment (Quiz 5% + Take-home 5%) | 10% |
| Midterm Exam | 30% |
| Final Exam | 60% |
